# 5.4 (實作) 串接「RY Tools」物流 (7-11/全家超商取貨)

我們不使用綠界的物流，而是使用「[RY Tools for WooCommerce](https://tw.wordpress.org/plugins/ry-woocommerce-tools/)」這個外掛，專門處理「**超商取貨**」（7-ELEVEN 和 全家 FamilyMart）。

**為什麼選擇 RY Tools？（2024-2025 年主流方案）**

* **彈性：** 它只專注做「物流」，不綁定「金流」。
* **體驗：** 顧客在結帳時，可以點選按鈕，跳出「**超商電子地圖**」，直接在地圖上選擇他要取貨的門市。
* **自動化：** 支援後台列印寄件單、自動更新物流狀態。

#### 步驟 1：安裝「[RY Tools for WooCommerce](https://tw.wordpress.org/plugins/ry-woocommerce-tools/)」

1. 回到您的 WordPress 後台。
2. 前往「外掛」>「安裝外掛」。
3. 搜尋「[RY Tools for WooCommerce](https://tw.wordpress.org/plugins/ry-woocommerce-tools/)」。
4. 安裝並「**啟用**」它。

#### 步驟 2：設定 RY Tools

1. 啟用後，在左側選單會出現「**RY Tools**」，點選它。
2. 在「一般設定」中，填寫您的「**寄件人姓名**」和「**寄件人手機**」。
3. 儲存設定。

#### 步驟 3：將「超商物流」加入「運送區域」

**這一步是銜接 `3.2` 章節的內容。**

1. 前往「WooCommerce」>「設定」>「**運送 (Shipping)**」頁籤。
2. 找到您在 `3.2` 建立的「**台灣本島**」運送區域，點選「**編輯**」。
3. \[示意圖：「台灣本島」區域的設定畫面]
4. 在「運送方式」的表格中，點選「**新增運送方式**」。
5. 會跳出一個小視窗。在下拉選單中，您會發現**多了 4 個新選項**：
   * `RY 7-ELEVEN (C2C)` (7-11 C2C 純取貨/取貨付款)
   * `RY FamilyMart (C2C)` (全家 C2C 純取貨/取貨付款)
   * `RY 7-ELEVEN (B2C)` (7-11 B2C 大宗寄倉)
   * `RY FamilyMart (B2C)` (全家 B2C 大宗寄倉)
6. **專家提示 (B2C vs C2C)：**
   * **C2C (個人對個人)：** 適合「**剛起步**」的賣家。您（個人 C）拿著包裹去超商櫃台寄給顧客（個人 C）。
   * **B2C (企業對個人)：** 適合「**貨量大**」的賣家。您（企業 B）必須先跟超商物流簽約（例如：大智通、日翊），您不需要去櫃台，物流車會每天來您的倉庫收貨。
   * **本課程建議：** 先啟用 **C2C**，這是最快、最沒有門檻的方案。
7. **新增 7-11 (C2C)：**
   * 選擇「**RY 7-ELEVEN (C2C)**」，點選「新增運送方式」。
   * 新增後，點選「**編輯**」。
   * **標題：** 輸入「**7-ELEVEN 超商純取貨**」。
   * **費用 (Cost)：** 輸入 `65` (或您設定的運費)。
   * **付款方式綁定：** (關鍵) **不要**勾選「啟用取貨付款」。我們假設這只是「純取貨」（顧客已用信用卡/LINE Pay 付款）。
   * **儲存設定**。
8. **新增 全家 (C2C)：**
   * 重複上述動作，選擇「**RY FamilyMart (C2C)**」。
   * **標題：** 輸入「**全家超商純取貨**」。
   * **費用 (Cost)：** 輸入 `65`。
   * **儲存設定**。

**結帳頁面變化：** 設定完成後，顧客在「結帳頁面」的「運送方式」區塊，就會看到：

* `[ ] 黑貓宅配 (本島) - $120`
* `[ ] 7-ELEVEN 超商純取貨 - $65`
* `[ ] 全家超商純取貨 - $65`
* `[ ] Moksa 工作室到店自取 - $0`
* (如果滿 $2000，還會多一個「消費滿 $2,000 免運費」選項)

當顧客選擇「7-ELEVEN」時，頁面會自動跳出一個「**選擇門市**」的按鈕，點擊即可開啟電子地圖。
