# 第 5 章：(重點) 台灣在地化金流與物流 (建議 3 小時)

前面幾章，我們已經學會了如何「蓋好房子」（架設網站）和「擺設商品」（上架商品）。**本章節的目標，是為您的房子裝上「收銀機」和「出貨系統」**。

在台灣經營電商，您必須串接符合本地消費者習慣的「金流」（信用卡、LINE Pay、超商付款）、「物流」（超商取貨、宅配）以及「電子發票」。

這章課的價值，就在於教您如何串接 **2024-2025 年**最新、最穩定的主流服務。

#### 在本章節，您將會學到：

1. **電商生態解析：** 為什麼我們（Moksa）推薦使用「金物流脫鉤」的專業架構？
2. **(實作) 綠界 ECPay 金流：** 如何申請並串接台灣市佔率最高的金流服務（信用卡、ATM）。
3. **(實作) LINE Pay 金流：** 如何串接市佔率最高的「行動支付」，大幅提升手機結帳轉換率。
4. **(實作)** [RY Tools for WooCommerce](https://tw.wordpress.org/plugins/ry-woocommerce-tools/)**：** 如何串接 7-11 與全家超商取貨，並實現「LINE Pay 付款、超商取貨」的彈性組合。
5. **(實作) 綠界 ECPay 電子發票：** 如何在 WooCommerce 串接電子發票，並設定「自動開立」時機。
6. **(實務) 結帳流程測試：** 從學員（顧客）視角，完整走一次下單、付款、收貨的流程，確保所有環節都正確運作。

完成本章後，您的網站將成為一個功能完整、可 24 小時自動收款、自動開發票的專業電商平台。
