# 5.2 (實作) 串接「綠界 ECPay」金流 (信用卡、ATM)

**注意：** 申請金流服務**必須**具備「**公司**」或「**商行**」的營業登記。

#### 步驟 1：申請綠界科技「特約會員」

1. 前往「綠界科技」官方網站。
2. 註冊並申請成為「**特約會員**」（B2C）。
3. 依照綠界指示，上傳您的「營業登記」、「身分證」、「銀行帳戶」等資料。
4. 等待綠界審核通過（可能需要 3-5 個工作天）。

#### 步驟 2：取得 3 組關鍵 API 金鑰

審核通過後，登入您的「綠界廠商後台」。

您必須找到並複製以下 3 組「**介接**」用的金鑰（請妥善保管）：

1. **商店代號 (MerchantID)**
   * (路徑：廠商後台 > 系統開發管理 > 系統介接設定)
2. **HashKey**
   * (路徑：廠商後台 > 系統開發管理 > 系統介接設定)
3. **HashIV**
   * (路徑：廠商後台 > 系統開發管理 > 系統介接設定)

\[示意圖：綠界後台的系統介接設定頁面，顯示這三組金鑰]

#### 步驟 3：安裝「綠界科技」官方外掛

1. 回到您的 WordPress 後台。
2. 前往「外掛」>「安裝外掛」。
3. 搜尋「**綠界科技 WooCommerce 金物流**」(ECPay Integration for WooCommerce)。
4. 安裝並「**啟用**」它。

#### 步驟 4：設定 ECPay「金流」

1. 前往「WooCommerce」>「設定」>「**付款 (Payments)**」頁籤。
2. 您會看到列表多了很多「綠界」的選項。
3. \[示意圖：付款頁籤中，綠界 ECPay 的信用卡選項]
4. 找到「**綠界 ECPay 信用卡**」，點選右側的「**管理 (Manage)**」按鈕。

#### 步驟 5：填入 3 組金鑰並啟用

1. **啟用/停用：** **勾選「啟用綠界 ECPay 信用卡」**。
2. **標題：** 輸入「信用卡 (一次付清)」（這是給顧客看的名稱）。
3. **商店代號 (MerchantID)：** 貼上您在步驟 2 取得的 `MerchantID`。
4. **HashKey：** 貼上您在步驟 2 取得的 `HashKey`。
5. **HashIV：** 貼上您在步驟 2 取得的 `HashIV`。
6. **自動請款：** 建議選擇「**是 (Yes)**」。(顧客刷卡成功後，綠界會自動向銀行請款，您才拿得到錢)。
7. 點選「**儲存設定**」。

#### 步驟 6：(重複) 啟用 ATM 與超商代碼

1. 回到「WooCommerce」>「設定」>「**付款 (Payments)**」頁籤。
2. 找到「**綠界 ECPay ATM**」，點選「**管理**」。
3. **啟用/停用：** **勾選「啟用綠界 ECPay ATM」**。
4. **標題：** 輸入「網路 ATM 轉帳」。
5. **金鑰：** ECPay 外掛（2024年以後的版本）通常會自動抓取您在信用卡設定中填入的金鑰，您**不需**重複填寫。如果需要，就再貼一次。
6. **儲存設定**。
7. **超商代碼 (CVS Code)** 也是重複一樣的動作。

**專家提示：**

* **測試模式：** 在綠界外掛的「一般設定」中，可以開啟「測試模式 (Test Mode)」。在網站「正式上線」對外營運前，**請務必關閉「測試模式」**，並使用您「**正式**」的 MerchantID、HashKey、HashIV，否則顧客刷的錢會是測試用的假資料！
* **SSL 憑證：** 如 `1.3` 所述，**您的網站必須有 `https` (SSL 憑證)**，否則綠界金流會串接失敗。
