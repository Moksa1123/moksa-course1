# 第 2 章：核心內容管理 (CMS) (建議 1.5 小時)



* **文章 (Post)：** 用於發布「最新消息」、「部落格」。
* **頁面 (Page)：** 用於建立「關於我們」、「聯絡我們」等固定資訊。

本章節的目標，是讓您**精通**這兩種內容的建立、編輯與管理，並學會如何處理網站的所有圖片（媒體庫）。

#### 在本章節，您將會學到：

1. 釐清「文章」與「頁面」的根本差異，從此不再搞混。
2. **(實作)** 從 0 到 1 發表一篇圖文並茂的「最新消息」。
3. **(實作)** 找到並編輯「關於我們」頁面中的文字。
4. **精通「媒體庫」**，包含如何上傳圖片、填寫 SEO 用的「替代文字」，以及如何「取代」舊圖片。

完成本章後，您將有能力獨立更新您網站上 90% 的靜態內容。
