5.5 如何產生「網址 (URL)」自動套用優惠

5.5 How to Create a "URL Coupon"

這是一個降低顧客使用門檻的絕佳功能。
您可以產生一個專屬網址，當顧客點擊這個網址時，優惠券會自動套用到他們的購物車，顧客完全不需要手動輸入代碼。

情境： 適用於 EDM 電子報、LINE@ 推播、或 Facebook 廣告。

步驟 1：建立優惠券

先依照一般流程，建立好一張您想要的優惠券（例如：SAVE100）。

將它「發表」或「更新」。

步驟 2：啟用 URL 優惠券

在該優惠券的編輯頁面，切換到「URL Coupons」頁籤。

勾選「啟用此優惠券的 URL」。

Code (代碼): (選填) 您可以自訂網址中的代碼，留空會使用優惠券代碼。

Override Link (覆蓋連結):

(選填) 這是最常用的功能之一。您可以設定顧客使用優惠券後，要被導向到哪個頁面。

範例： 填入您的「商店頁面」網址。這樣顧客一點連結，就會自動套用優惠，並立刻被導向商店頁面開始購物。

Add Products to Cart?

(選填) 您甚至可以設定當顧客點此連結時，自動將特定商品加入購物車。這在 BOGO 活動中非常有用。

步驟 3：取得並使用網址

點選「更新」儲存優惠券。

再次回到「URL Coupons」頁籤。

您會看到系統在 Coupon URL 欄位產生了一組專屬網址。

複製這串網址。

現在，您可以將這串網址貼到您的 Facebook 貼文、EDM 或 LINE 訊息中。顧客點擊後，就會自動套用折扣。

延伸應用 (QR Code)

您可以將這串「專屬網址」拿去任何 QR Code 產生器（網路上很多免費的），將它轉換成一個 QR Code。
然後將這個 QR Code 印在您的實體文宣或名片上。顧客用手機一掃，就能自動套用優惠！
