# 第 4 章：(重點) 商品管理系統 (建議 3 小時)

WooCommerce 提供了強大的商品系統，可以讓您販售任何類型的商品。

#### 在本章節，您將會學到：

1. **商品分類策略：** 如何規劃「分類」(Category) 與「標籤」(Tag)，讓顧客更容易找到商品。
2. **(實作) 簡單商品：** 如何上架「沒有」規格選項的商品（例如：一本書、一個馬克杯）。
3. **(實作) 可變商品：** 如何上架「有」規格選項的商品（例如：一件 T-shirt 有**不同顏色**和**不同尺寸**），這是 WooCommerce 最強大也最關鍵的功能。
4. **其他商品類型：** 如何販售「服務」(虛擬商品)、「電子書」(可下載商品)、「組合包」(分組商品) 與「導購商品」(聯盟商品)。
5. **(實務) 庫存管理：** 深入了解「單品庫存」與「全域庫存」的設定差異。
6. **(實務) SEO 應用：** 如何利用「屬性」和「標籤」來提升您商店的 SEO 排名。

完成本章後，您將有能力將您所有的產品，無論多複雜，都能有條理且專業地在您的網站上架販售。
