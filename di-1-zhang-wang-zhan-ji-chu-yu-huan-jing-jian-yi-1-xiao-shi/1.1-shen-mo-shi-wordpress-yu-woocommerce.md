# 1.1 什麼是 WordPress 與 WooCommerce？

#### 什麼是 WordPress？

簡單來說，**WordPress 是一個「網站的作業系統」**。

* 就像您的筆記型電腦需要安裝 Windows 或 macOS 才能運作。
* 您的網站也需要一個「作業系統」來管理所有內容，例如：發表文章、上傳圖片、管理頁面。
* WordPress 就是全世界最受歡迎的那個作業系統，它是一個「內容管理系統」(CMS, Content Management System)。

**一個實用的比喻：樂高 (LEGO)** 我們常把 WordPress 比喻為「樂高」：

* **WordPress 核心 (Core)：** 就像是那塊綠色的大底板，它提供了最基礎的功能。
* **佈景主題 (Theme)：** 就像是樂高的「外觀包」（例如：城堡系列、太空系列）。它決定了您網站的「**外觀**」和「風格」。
* **外掛 (Plugin)：** 就像是樂高的「特殊零件」（例如：輪子、窗戶、引擎）。您需要「購物車」功能嗎？安裝一個外掛。您需要「聯絡表單」嗎？再安裝一個外掛。

**為什麼選擇 WordPress？** 因為它「開源」(Open Source)、「彈性高」(高可塑性)、「社群強大」(不怕找不到資源)，並且「SEO 友善」(Google 喜歡的結構)。

#### 什麼是 WooCommerce？

如果 WordPress 是「樂高底板」，那 **WooCommerce 就是「一整組【超級商店】的樂高零件包」**。

* **WooCommerce** 本身是一個「**外掛 (Plugin)**」。
* 它是 WordPress 官方推出、全世界最多人使用的「電商外掛」。
* 一旦您在 WordPress 上安裝並啟用了 WooCommerce，您的網站就會立刻**從一個「部落格」升級為一個「功能完整的電子商務平台」**。
* 它會自動幫您建立：
  * **商品上架系統**
  * **購物車功能**
  * **結帳流程**
  * **訂單管理系統**
  * **金流/物流 串接介面**

**總結：**

* **WordPress** 是您的「網站本身」。
* **WooCommerce** 是讓您的網站「能夠賣東西」的「電商外掛」。
* 我們這 30 小時的課程，就是圍繞著這兩大核心展開的。
