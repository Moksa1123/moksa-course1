# 1.2 (關鍵) 釐清 WordPress.org 與 WordPress.com

| 比較項目     | WordPress.org (本課程)                                                                        | WordPress.com (類似痞客邦/Wix)                                |
| -------- | ------------------------------------------------------------------------------------------ | -------------------------------------------------------- |
| **比喻**   | **您買下一塊地，自己蓋房子**                                                                           | **您去租一間「社區公寓」**                                          |
| **彈性**   | **極高**。您擁有 100% 控制權。                                                                       | **極低**。您被綁在房東的規則裡。                                       |
| **外掛**   | <p>✔️ <strong>可安裝 (全部)</strong></p><p>（這就是彈性的來源）</p>                                       | <p>❌ <strong>不可安裝</strong> (免費/平價方案)</p><p>（高價方案才開放）</p> |
| **佈景主題** | <p>✔️ <strong>可安裝 (全部)</strong></p><p>（包含您自己買的、客製的）</p>                                    | <p>❌ <strong>不可安裝</strong> (免費/平價方案)</p><p>（只能用它內建的）</p> |
| **營利**   | <p>✔️ <strong>完全自由</strong></p><p>（您可放廣告、做電商）</p>                                          | <p>❌ <strong>限制</strong></p><p>（免費版會被強迫放廣告）</p>          |
| **費用**   | <p>1. <strong>主機費 (必要)</strong></p><p>2. <strong>網域費 (必要)</strong></p><p>3. 外掛/佈景 (選用)</p> | <p>1. <strong>月租費 (必要)</strong></p><p>（從免費到高價方案）</p>     |
| **適合誰**  | **專業經營者、電商、企業**                                                                            | 純粹想寫部落格的作家                                               |

**課程總結：**

* **`.com`** 像是一個「**服務**」（例如痞客邦 Pixnet），您付錢給他，他幫您搞定一切，但您也被他綁死。
* **`.org`** 是一個「**軟體**」（就像 Office 軟體），您免費下載它，然後安裝在您自己的「主機」上，您擁有 100% 的主導權。
* **本課程 100% 只專注在 WordPress.org**，因為這才是能真正「經營事業」的專業工具。
