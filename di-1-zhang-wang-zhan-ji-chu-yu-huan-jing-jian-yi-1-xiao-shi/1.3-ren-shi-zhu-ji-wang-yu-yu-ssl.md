# 1.3 認識主機、網域與 SSL

#### 1. 主機 (Hosting)

* **比喻：您買的「土地」**。
* **定義：** 主機是一台（或一組）24 小時開機、連接著高速網路的「超級電腦」（伺服器, Server）。您的網站所有檔案、圖片、文章、資料庫... 全部都存放在這台電腦上。
* **重要性：**
  * **土地的「地段」(速度)：** 好的主機（例如：Cloudways, Kinsta）通常費用較高，但能讓您的網站「秒開」，顧客才不會沒耐心就關掉。
  * **土地的「大小」(空間)：** 決定了您可以放多少商品圖片、多少文章。
  * **土地的「管理員」(安全性)：** 好的主機商會提供防火牆、自動備份，保護您的網站（您的房子）不被駭客入侵。

#### 主機類型大解密 (為何價格差這麼多？)

這是經營者最常問的問題：「為什麼有 100 元/月的主機，也有 3000 元/月的主機？」

| 主機類型    | 虛擬主機 (Shared Hosting)                                   | VPS (虛擬專屬主機)                                     | 雲端主機 (Cloud Hosting) (本課程推薦)                       |
| ------- | ------------------------------------------------------- | ------------------------------------------------ | -------------------------------------------------- |
| **比喻**  | **學生宿舍 (多人合租)**                                         | **獨立套房**                                         | **五星級飯店 (隨選服務)**                                   |
| **速度**  | <p><strong>慢、不穩定</strong></p><p>（您的鄰居（網站）半夜開派對會吵到您）</p> | <p><strong>快</strong></p><p>（您有自己的獨立空間）</p>      | <p><strong>極快、彈性</strong></p><p>（資源隨時可擴充）</p>      |
| **資源**  | <p><strong>共享</strong></p><p>（CPU/RAM 所有人共用）</p>        | <p><strong>保證</strong></p><p>（您有固定的 CPU/RAM）</p> | <p><strong>可隨時擴充</strong></p><p>（本月活動流量大？一鍵升級）</p> |
| **管理**  | 簡單 (主機商管理)                                              | **極困難** (您要自己當網管)                                | **簡單** (例如 Cloudways 幫您管理)                         |
| **價格**  | **極便宜** (NT$ 100-300 / 月)                               | 中等 (NT$ 500-1500 / 月)                            | 中高 (NT$ 1000-3000+ / 月)                            |
| **適合誰** | 測試、作品集、沒預算                                              | 懂技術的工程師                                          | **專業電商、企業用戶**                                      |

**專家總結：**

* **絕對不要用「虛擬主機」來做電商**。您的鄰居網站如果被駭，您的網站也會跟著遭殃；您的鄰居網站如果流量暴增，您的網站會跟著卡死。
* 我們推薦使用\*\*「雲端主機」\*\*（例如：Cloudways, Kinsta），雖然較貴，但它提供了電商網站最重要的「**速度**」、「**穩定**」和「**彈性**」。在主機上省錢，絕對是因小失大。

#### 2. 網域 (Domain Name)

* **比喻：您家的「門牌地址」**。
* **定義：** 一個獨一無二、用來找到您網站的「名字」。例如 `moksaweb.com` 或 `google.com`。
* **重要性：**
  * 它是您在網路上的「品牌」。
  * 顧客是透過「網域」來找到您的「主機」（土地）。
  * 網域是需要「**每年**」付費租用的（向 GoDaddy, Namecheap, 或是中華電信等網域註冊商）。

#### 3. SSL 憑證 (SSL Certificate)

* **比喻：您家門口的「高級門鎖」**。
* **定義：** 一種「安全加密憑證」。
* **您怎麼看出來？**
  * **有 SSL：** 網址開頭是 `https`:// (多一個 S)，瀏覽器會顯示一個「**鎖頭**」圖案。
  * **沒有 SSL：** 網址開頭是 `http`://，瀏覽器會顯示「**不安全**」的警告。
* **重要性：**
  1. **保護顧客資料：** 當顧客在您的網站上輸入信用卡號、密碼、個資時，SSL 會將這些資料「加密」，防止在傳輸過程中被駭客竊取。
  2. **信任感：** 顧客看到「不安全」的警告，根本不敢在您的網站下單。
  3. **SEO 必需品：** Google 已公開聲明，**沒有 SSL 的網站，SEO 排名會被嚴重扣分**。
  4. **金流必需品：** 台灣的綠界、藍新等金流服務商，**強制**要求您的網站必須安裝 SSL 才能串接。

**總結：** 我們會租用一塊高品質的「**雲端主機**」(土地)，在上面蓋好 WordPress 這棟房子，然後去申請一個「**網域**」(地址) 掛在門口，最後裝上「**SSL**」(門鎖) 保護訪客與金流安全。
