9.3 Website Security (Wordfence)

您的網站安裝了「Wordfence Security」外掛，它就像是您網站的 7-24 警衛和防火牆。

Moksa 團隊已經為您設定好基礎的防護，您不需要做複雜的操作，但您需要了解它在做什麼。

1. 防火牆 (Firewall)

Wordfence 會自動阻擋「已知」的惡意 IP、機器人，防止它們嘗試猜測您的登入密碼（稱為「暴力破解攻擊」）。

您不需要操作，它會在背景自動保護您。

2. 惡意軟體掃描 (Scan)

Wordfence 會定期自動掃描您網站的所有檔案，比對看是否有「惡意程式碼」或「後門」被植入。

您需要做的：

在左側選單，點選「Wordfence」>「Scan」。

定期檢查掃描結果。 理想情況下，您應該會看到「Congratulations! No security issues were found.」（恭喜！未發現安全問題）的綠色訊息。

如果它跳出紅色或黃色的警告，告訴您某個檔案有問題，請不要慌張，也不要自己刪除。

立即截圖，並通知 Moksa 團隊，讓我們來為您處理。

3. 登入安全

Wordfence 會監控登入行為。如果您（或任何人）連續登入失敗太多次（例如：5 次），Wordfence 會暫時封鎖您的 IP，以防止暴力破解。

如果您發現自己被封鎖了：

請稍等 15-30 分鐘再試。

或者，切換到手機 4G 網路（更換 IP）再登入。

或者，直接聯繫 Moksa 團隊為您解鎖。
