1. 歡迎使用

1. Welcome

歡迎使用 Moksa 團隊為您建置的 WordPress 網站！

這本操作手冊是專為您設計的，目的是幫助您在我們的教學課程結束後，仍能隨時回來查閱網站的各項操作功能。我們將所有功能都拆解成詳細的步驟，並附上截圖說明，讓您可以安心地管理您的網站。

如何使用這本手冊？

使用左側目錄： 您可以透過左側的導覽列，快速跳轉到您想查找的功能，例如「如何新增商品」或「如何設定優惠券」。

使用上方搜尋： GitBook 提供強大的搜尋功能。如果您不確定功能在哪個章節，可以直接在最上方的搜尋框輸入關鍵字（例如：「紅利點數」），就能找到所有相關的頁面。

遇到問題怎麼辦？

如果您在這本手冊中找不到答案，或在操作時遇到任何困難，請隨時透過我們雙方的聯絡管道（例如 LINE）與 Moksa 團隊聯繫，我們將盡快為您提供協助。

在開始之前，請先閱讀下一個章節，學習如何登入網站的後台。
