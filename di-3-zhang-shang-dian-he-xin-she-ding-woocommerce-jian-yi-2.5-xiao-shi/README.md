# 第 3 章：商店核心設定 (WooCommerce) (建議 2.5 小時)

網站架設時，這些功能通常會被預先設定好，但身為經營者，您必須了解這些設定的意義，因為它們**直接影響您的營收、物流成本與法律合規性**。

**本章節的目標，是讓您學會檢查與調整所有商店營運的幕後細節。**

#### 在本章節，您將會學到：

1. **基礎設定：** 設定您的「商店地址」、「幣別」（設為 `TWD`）。
2. **(實作) 運送方式：** 精通「**運送區域**」邏輯，設定出「**本島宅配**」、「**外島運費**」、「**到店自取**」以及「**免運門檻**」。
3. **(實作) 稅金設定：** 如何設定 5% 營業稅，以及處理「**含稅/未稅**」與「**免稅商品**」的情境。
4. **商品設定：** 管理「商品評論」與「商店頁面」的顯示方式。
5. **庫存設定：** 設定「低庫存通知」，讓您不再超賣。
6. **(實務) 電子郵件：** 如何「**客製化**」寄給客戶的訂單通知信（例如：加入公司 Logo）。
7. **帳號與隱私權：** 包含「訪客結帳」與「隱私權政策」的法規設定。

完成本章後，您就完成了商店的基礎建設，準備好開始上架商品了！
